# Введение

**ClientSamgk** это Dotnet библиотека для работы с
REST API [Самарского государственного колледжа](https://samgk.ru/).

## Установка

Библиотека доступна в [nuget](https://www.nuget.org/packages/ClientSamgk)
и в виде [.nupkg файла](https://github.com/TheCrazyWolf/Api-Client-for-samgk.ru/releases/latest).

Установите в проект при помощи Dotnet CLI:
```cmd copy
dotnet add package ClientSamgk
```
