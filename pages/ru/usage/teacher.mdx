# API Преподаватели

### Получить список всех преподавателей
```Accounts.GetTeachers``` вернет список всех имеющихся преподавателей
<Tabs items={['Синхронно', 'Асинхронно']}>
  <Tabs.Tab>
    ```csharp
    var teachers = api.Accounts.GetTeachers();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```csharp
    var teachers = await api.Accounts.GetTeachersAsync();
    ```
  </Tabs.Tab>
</Tabs>

### Поиск конкретного преподавателя
```Accounts.GetTeacher``` попробует найти конкретного преподавателя по ФИО

    ```csharp

    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```csharp
    var teacher = await api.Accounts.GetTeacherAsync("Иванов Иван Иванович");
    ```
  </Tabs.Tab>
</Tabs>
