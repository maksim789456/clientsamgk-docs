# API Кабинеты / Корпуса

### Получить список всех кабинетов
```Cabs.GetCabs``` вернет список всех имеющихся кабинетов
<Tabs items={['Синхронно', 'Асинхронно']}>
  <Tabs.Tab>
    ```csharp
    var cabs = api.Cabs.GetCabs();
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```csharp
    var cabs = await api.Cabs.GetCabsAsync();
    ```
  </Tabs.Tab>
</Tabs>

#### Получить список всех кабинетов для конкретного корпуса
Можно ограничить выборку, если явно передать номер корпуса
<Tabs items={['Синхронно', 'Асинхронно']}>
  <Tabs.Tab>
    ```csharp
    var cabsOnlyIn5 = api.Cabs.GetCabs("5");
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```csharp
    var cabsOnlyIn5 = await api.Cabs.GetCabsAsync("5");
    ```
  </Tabs.Tab>
</Tabs>
